# Future Digest – User Manual

Welcome to the official User Manual for Digest
This document explains how to access, use, and troubleshoot the system.

---

## 1. Overview

Digest is an interactive web-based data visualization built using D3.js and Scrollama. It allows users to explore how their daily diet routines may change in the next 50 years using current climate and agicultural predications.

The system is designed for general users and requires no technical knowledge to operate.

---

## 2. System Requirements

To use this project, you will need:

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- JavaScript enabled
- Recommended screen width: 1200px or larger
- Stable internet connection (if using hosted version)

## 4. FAQ

Q: Where are these projections coming from?
A: The projections for this project are aggredated from a number of different data sets published the United Nations, MIT, and more. The full list will be available upon completion of the project.

Q: What if I inputed an incorrect value for my food?
A: The visualization can be reset by pressing the reset button in the top right. Otherwise, feel free to just explore!

Q: What if I disagree with the data presented to me?
A: The data presented is just a predication. It is not a garunteed future nor would it be accepted by all experts.

Q: I find the information presented in this visualization alarming. What should I do?
A: The purpose of this visualization is to present the ways that climate change will change our daily lives in a way that does induce panic. However, it can be difficult to ingest this information without feeling unease. Just know that while you as an individual has an impact on the environment, the cause of climate change is structure of our society and not an one person.
